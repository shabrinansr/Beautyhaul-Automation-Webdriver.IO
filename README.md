Beautyhaul Automation with WebdriverIO
Welcome to the Beautyhaul Automation repository! This project contains a sample automation script created with WebdriverIO to test the Beautyhaul website.

🚀 Project Overview
This repository includes:
- Sample Automation Script: Developed with WebdriverIO to demonstrate automated testing for the Beautyhaul website.
- Basic Setup Guide: Instructions for preparing your environment to run the script smoothly.

🛠️ Prerequisites
Before running this script, make sure you have the following installed:
- Java - Required for WebdriverIO's underlying components.
- Visual Studio Code (or another preferred code editor) - For managing and running the script.
- Node.js and WebdriverIO - WebdriverIO needs to be installed for script execution.

⚠️ Troubleshooting
If you encounter errors, here are a few possible reasons:
- Website Updates: The Beautyhaul website may have undergone changes, which can cause discrepancies between the script and the site.
- Missing Dependencies: Ensure that all required software and dependencies are correctly installed before running the script.
